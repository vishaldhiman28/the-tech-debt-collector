Quick Start

Installation

# Clone repository
git clone https://github.com/yaegar/tech-debt-collector
cd tech-debt-collector

# Build binary
go build -o bin/tech-debt-collector ./cmd/tech-debt-collector

Basic Usage

# Analyze current directory
./tech-debt-collector -path .

# Enable LLM analysis (requires OpenAI key)
export OPENAI_API_KEY="sk-..."
./tech-debt-collector -path . -llm

# View report
cat report.json | jq .
